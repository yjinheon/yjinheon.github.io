Intro

Docker개념

  • 도커 이미지는 다른 이미지 위에 쌓는게 가능하다.
  • VM은 독립적으로 운영되고 독립적인 자원을 사용하기 때문에 그 경우에 효율적일 수 있다.
  • 도커의 Layer는 설계도이다.
  • 이미지는 기본적으로 아직 실행되지 않은 컨테이너이다.

Docker Container는 가상환경과 다르다.

가상환경은 python version만 폴더안에 있는걸 쓰는 것이고 Docker Container는 아예 독립된 환경이다.
