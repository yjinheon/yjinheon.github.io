Author  published on 2021-07-07 included in  category Categories
Docker개념 도커 이미지는 다른 이미지 위에 쌓는게 가능하다. VM은 독립적으로 운영되고 독립적인 자원을 사용하기 때문에 그 경우에 효율적일 수 있다. 도커의 Layer는 설계도이다. 이미지는 기본적으로 아직 실행되지 않은 컨테이너이다. Docker Container는 가상환경과 다르다. 가상환경은 python version만 폴더안에 있는걸 쓰는 것이고 Docker Container는 아예 독립된 환경이다. Doker 명령어들도커 빌드하기 1$ cd /path/to/Dockerfile 2$ sudo docker build . 현재 실행되고 있는 도커 프로세스 확인 1$ sudo docker ps 전체 도커 프로세스 확인
