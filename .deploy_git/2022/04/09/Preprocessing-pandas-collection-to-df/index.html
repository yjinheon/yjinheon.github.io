Python으로 데이터 분석을 하다보면 pandas를 찾게 되는 경우가 많다.
Python 기본 자료형을 pandas에서 제공하는 자료형으로 변환하는 데 익숙해지면 전처리작업을 보다 수월할게 할 수 있다.

여기서는 작업을 하면서 자주쓰게 되는 자료형 변환 방법들을 정리하였다
